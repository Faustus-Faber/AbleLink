# AbleLink User Guide

## Getting Started
AbleLink is designed to be fully accessible. You can navigate using:
- Mouse & Keyboard
- Screen Readers (NVDA, JAWS)
- Voice Commands (Enable in Accessibility Settings)
- Agent Assistance ("AbleBot")

## Finding a Job
1. Go to the "Find Jobs" page.
2. Filter by "Accessibility Features" (e.g., Screader Compatible, Wheelchair Access).
3. Click "Apply Now" to submit your profile.
4. Employers will contact you via the Messaging system.

## Learning Hub
Our courses include:
- Subtitles and Transcripts for all videos.
- Sign Language Interpretation (select courses).
- Easy-Read summaries.

## Health Tools
For disabled users, the Health Dashboard tracks:
- Medication schedules (with reminders).
- Upcoming doctor appointments.
- Daily vitals (log manually or sync).
Caregivers can also access this data if authorized.
